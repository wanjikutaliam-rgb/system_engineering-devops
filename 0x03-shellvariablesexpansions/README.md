# System Engineering & DevOps Shell Projects

This repository contains shell scripting exercises for the System Engineering and DevOps course. The tasks cover **shell redirections, aliases, variables, and expansions**.

## Repository Structure

- `0x02-shell_redirections/`  
  Contains scripts that demonstrate the use of shell redirections such as output, input, appending, and file handling.

- `0x03-shell_variables_expansions/`  
  Contains scripts that demonstrate aliases, environment variables, and shell expansions.
