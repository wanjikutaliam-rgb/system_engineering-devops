Project Overview

This project focuses on mastering loops, conditions, and parsing in Bash scripting. Learners will gain hands-on experience with repetitive operations, conditional logic, and parsing input/data to automate tasks in a Linux shell environment.

By completing these tasks, learners will understand how to:

Use for, while, and until loops.

Implement conditional statements (if, else, elif, case).

Parse and manipulate input or file content using Bash tools.

Tasks
1. For Best School Loop

File: 1-for_best_school

Objective: Display the phrase "Best School" 10 times using a for loop.

Requirements:

Must use a for loop only.

while and until loops are forbidden.

Example Output:

Best School
Best School
Best School
Best School
Best School
Best School
Best School
Best School
Best School
Best School

How to Run

Open a terminal in the project directory.

Make the script executable:

chmod +x 1-for_best_school


Execute the script:

./1-for_best_school

Notes

The first line of all scripts should be:

#!/usr/bin/env bash


Include a comment explaining the script purpose on the second line.

Ensure all outputs match the examples provided for automated checking.
