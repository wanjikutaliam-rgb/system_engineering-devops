# 0x00-shell_basics

This directory contains basic shell scripts designed to help learners get familiar with Linux command-line operations and file system navigation. Each script addresses a specific task related to understanding directories, files, and fundamental shell commands.

## Files and Tasks

- **0-current_working_directory**: Prints the absolute path of the current working directory.
